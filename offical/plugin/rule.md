# 规范
## [@efox/eslint-config-react](https://github.com/efoxTeam/emp/tree/main/packages/eslint-config-react)
+ 提供 react typescript eslint 支持
## 安装
```
pnpm add @efox/eslint-config-react
```
## 配置
> 根目录创建 `.eslintrc.js`
```js
module.exports = {
  extends: ['@efox/eslint-config-react'],
}
```
